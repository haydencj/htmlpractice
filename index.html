More about HTML images:

HTML images are defined with the img tag. The source file (src), alternative text (alt) are required attributes, but it's advised you also include the width and height attributes.

There are two ways to specify the URL in the src attribute. Absolute URL, links to an external image hosted on another websote. Relative URL, links to an image that is hosted within the website. check above img tag for example.
